# Tailwind Market Kit

> Digital Marketplace Kit for Tailwind

Install the plugin from npm:

```
$ npm install digimarket-kit
```

Then add the plugin to your `tailwind.config.js` file:

```

export default {
    plugins: [
        require('digimarket-kit/plugin')
    ]
}

```
